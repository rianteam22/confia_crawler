#  16.2. Minimizing `grind` calls[🔗](find/?domain=Verso.Genre.Manual.section&name=The-Lean-Language-Reference--The--grind--tactic--Minimizing--grind--calls "Permalink")
The `grind only [...]` tactic invokes `[grind](Tactic-Proofs/Tactic-Reference/#grind "Documentation for tactic")` with a limited set of theorems, which can improve performance. Calls to `grind only` can be conveniently constructed using `[grind?](Tactic-Proofs/Tactic-Reference/#grind___ "Documentation for tactic")`, which automatically records the theorems used by `[grind](Tactic-Proofs/Tactic-Reference/#grind "Documentation for tactic")` and suggests a suitable `grind only`.
These theorems will typically include a symbol prefix such as `=`, `←`, or `→`, indicating the pattern that triggered the instantiation. See the [section on E-matching](The--grind--tactic/E___matching/#e-matching) for details. Some theorems may be labelled with a `usr` prefix, which indicates that a custom pattern was used.
