#  12.3. Multi-Threaded Execution[🔗](find/?domain=Verso.Genre.Manual.section&name=The-Lean-Language-Reference--Run-Time-Code--Multi-Threaded-Execution "Permalink")
Lean includes primitives for parallel and concurrent programs, described using [tasks](IO/Tasks-and-Threads/#--tech-term-Tasks). The Lean runtime system includes a task manager that assigns hardware resources to tasks. Along with the API for defining tasks, this is described in detail in the [section on multi-threaded programs](IO/Tasks-and-Threads/#concurrency).
