#  19. Basic Propositions[🔗](find/?domain=Verso.Genre.Manual.section&name=basic-props "Permalink")
With the exception of implication and universal quantification, logical connectives and quantifiers are implemented as [inductive types](The-Type-System/Inductive-Types/#--tech-term-Inductive-types) in the `Prop` universe. In some sense, the connectives described in this chapter are not special—they could be implemented by any user. However, these basic connectives are used pervasively in the standard library and built-in proof automation tools.
