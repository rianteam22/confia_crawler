#  Supported Platforms[🔗](find/?domain=Verso.Genre.Manual.section&name=platforms "Permalink")
##  Tier 1[🔗](find/?domain=Verso.Genre.Manual.section&name=The-Lean-Language-Reference--Supported-Platforms--Tier-1 "Permalink")
Tier 1 platforms are those for which Lean is built and tested by our CI infrastructure. Binary releases of Lean are available for these platforms via [`elan`](Build-Tools-and-Distribution/Managing-Toolchains-with-Elan/#elan). The Tier 1 platforms are:
  * `x86-64` Linux with glibc 2.26+
  * `aarch64` Linux with glibc 2.27+
  * `aarch64` (Apple Silicon) macOS 10.15+
  * `x86-64` Windows 11 (any version), Windows 10 (version 1903 or higher), Windows Server 2022, Windows Server 2025


##  Tier 2[🔗](find/?domain=Verso.Genre.Manual.section&name=The-Lean-Language-Reference--Supported-Platforms--Tier-2 "Permalink")
Tier 2 platforms are those for which Lean is cross-compiled but not tested by our CI. Binary releases are available for these platforms.
Releases may be silently broken due to the lack of automated testing. Issue reports and fixes are welcome.
The Tier 2 platforms are:
  * `x86-64` macOS 10.15+
  * Emscripten WebAssembly
